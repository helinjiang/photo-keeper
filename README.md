# photo-keeper

基于 electron 开发的照片管理助手

使用 [electron-react-boilerplate](https://github.com/electron-react-boilerplate/electron-react-boilerplate) 初始化项目。

## 开发调试

安装依赖：

```
yarn
```

启动调试：

```
yarn start
```
